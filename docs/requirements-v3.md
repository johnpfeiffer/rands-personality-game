# Goal

To build the MVP react SPA application that asks the users questions and then displays their resulting "best match" Personality 

## Requirements

- ask all of the questions from the question bank in a linear way
- store the score for each question and sum
- do allow the user to "restart from the beginning" at every step
- - restart resets all the scores
- do not allow the user to go back and modify answers (too much complexity for now)
- display of the personality should be an easy URL like "/wolf" 
- display of the personality should show all the scores once expanded
- - the result personality should have a link to the reference articles from Rands blog

## Design

- Please use a minimalist Light Mode approach, preferring ascii and text over images
- Mobile and flexible resizing friendly 
- Principle of "progressive disclosure" like making the extra info as (easy to find and click) expanders

## Implementation

- use Material-UI
- use React + TS
- follow the agent.md (be modular)
